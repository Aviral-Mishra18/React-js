# React.js Learning Journey

This repository tracks my progress while learning React.js. It contains various projects and exercises focused on building modern web applications.

## Projects

### 01-folder: Vite + React Starter
A basic React project initialized using Vite.

#### Features
- Fast development with Vite HMR (Hot Module Replacement)
- React functional components
- Port-stabilized configuration in `vite.config.js`

#### Getting Started
1. Navigate to the folder:
   ```bash
   cd 01-folder
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Tech Stack
- React
